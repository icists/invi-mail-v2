# invi-mail-v2
Google 스프레드시트와 Gmail과 연동한 자동 메일 매크로

### 사용법

#### 설치
우선 이 저장소를 clone하세요.
'''bash
git clone https://github.com/icists/invi-mail-v2.git
'''

그 다음 해당 디렉토리로 이동하고
'''bash
cd invi-mail-v2
'''
가상환경을 실행하세요.
'''bash
python -m venv .
cd Scripts
activate.bat
'''

이제 패키지를 설치합니다.
'''bash
cd ..
pip install -r requirements.txt
'''
